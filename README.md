# TicTacToe in Vue

This is a simple project used to show how the pen-and-pencil paper game TicTacToe can be implemented by relying 
on Vue.js.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
